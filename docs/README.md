## Active Direction
The active product is a buyer-trust search platform:
- Map-first search.
- Sold price visibility where licensed.
- Listing history and relisting timelines.
- Comparable sale tooling.
- Neighborhood and school context.
- Fast mobile listing detail pages.
- Explicit tour or agent-introduction requests only.

The previous agent AI co-pilot direction is superseded.

## Planning Priorities
Before implementation, resolve:
- First Canadian launch region.
- MLS/sold-data access path.
- Listing-photo and sold-data display rules.
- Map provider choice.
- Auth and managed database choice.

Implementation can begin with permitted seed/demo data while the data access path is negotiated.
